I just made one of my recent infrastructure projects public: Rule Engine Core.

- replay execution for deterministic validation
- stdout, file, webhook, queue, and object-storage sinks

## Add To LinkedIn

1. Open LinkedIn profile.
2. Click `Add profile section`.
3. Under `Recommended`, add `Projects`.
4. Enter the project details from this file.
5. In the project entry, use `Add media` then `Add a link` for the GitHub URL.
6. Under `Recommended`, add `Featured`.
7. Add the GitHub link and optionally upload `docs/architecture.svg` or a PNG export of it.
8. Publish the post copy above.
9. If the post performs well, add that post into `Featured` too.
